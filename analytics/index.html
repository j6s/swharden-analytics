Clean Analytics
Minimal privacy-focused website analytics

About

Clean Analytics is a system for logging page views across multiple websites and providing a dashboard with page analytics plots and queryable reports. It was created to be a simpler and privacy-focused alternative to Google Analytics.

Features

  • IP addresses are not stored in the database
  • Cookies are never used to track users
  • Page views can be viewed across multiple websites from a single dashboard
  • Tracking code can be used on any HTML page, including GitHub Pages
  • Desktop applications can send telemetry using HTTP requests to the logging API
  • Dashboard shows realtime streaming logs as pages are viewed
